# GradeSense — pinned dependencies for Streamlit Community Cloud
#
# Pin policy
# -----------
# Streamlit Cloud resolves requirements.txt from scratch on every
# deploy, so we pin to specific versions (or version ranges with
# an upper bound) known to work together. The `~=` operator pins
# to compatible releases (>= X.Y, == X.*) — bump the minor version
# after re-testing locally.
#
# Why these specific versions
# ----------------------------
# - streamlit 1.39+ supports `st.cache_resource` and the modern
#   `st.set_page_config(layout="wide")` API used in app/main.py.
# - pandas 2.x keeps the DataFrame API our pipeline uses.
# - numpy 1.26 is the last release that works with both scikit-learn
#   1.4 and shap 0.45 on Python 3.11; 2.x requires newer shap.
# - shap 0.45 is the last release that bundles a pickled TreeExplainer
#   compatible with the artifacts on disk (model/explainer.joblib).
# - scikit-learn 1.4 is what the pipeline was trained with; loading
#   a 1.4-trained joblib into a 1.5+ sklearn is fine, but the ColumnTransformer
#   attribute names changed in 1.5, so we keep the same major.minor.
#
# Reproducible build
# ------------------
# To re-resolve: `pip install -r requirements.txt --no-cache-dir`.
# To check for newer compatible releases: `pip list --outdated`.

# Web framework
streamlit>=1.39,<1.50

# Modeling
scikit-learn>=1.4,<1.6
joblib>=1.3,<2.0

# Explainability
shap>=0.45,<0.46

# Data
pandas>=2.1,<3.0
numpy>=1.26,<2.0

# Charts (Plotly is the only charting library actually used in
# app/main.py — matplotlib was the old matplotlib chart and is no
# longer imported anywhere).
plotly>=5.18,<7.0
